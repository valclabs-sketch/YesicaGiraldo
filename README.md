# Yesica Giraldo Aesthetics Website

Responsive static website starter for Yesica Giraldo Aesthetics, London.

## Files

- `index.html` — main page and SEO metadata
- `styles.css` — responsive visual design
- `script.js` — mobile menu and subtle scroll animations
- `robots.txt` — search-engine crawling rules
- `sitemap.xml` — starter sitemap
- `assets/` — logo and future photography

## Preview locally

Open `index.html` in a browser, or use the VS Code extension **Live Server**.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload all the files in this folder to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.
6. GitHub will provide a temporary `github.io` URL.

## Connect a custom domain

1. Buy the domain from a registrar such as Namecheap, GoDaddy, Cloudflare Registrar or IONOS.
2. In GitHub, open **Settings → Pages → Custom domain**.
3. Enter the domain, for example `yesicagiraldoaesthetics.co.uk`.
4. At the domain registrar, add the DNS records shown by GitHub.
5. Enable **Enforce HTTPS** once DNS propagation is complete.
6. Replace every instance of `https://www.yourdomain.co.uk/` in:
   - `index.html`
   - `robots.txt`
   - `sitemap.xml`
7. Add a file named `CNAME` containing only the final domain.

## Before launch

- Replace all image placeholders with optimised WebP/AVIF photographs.
- Replace placeholder reviews with genuine, authorised testimonials.
- Confirm treatment names and wording.
- Add the full postcode and verified opening hours.
- Replace the placeholder canonical and social-sharing URLs.
- Connect Google Search Console and submit `sitemap.xml`.
- Create or update the Google Business Profile with matching name, address and phone.
- Add a privacy policy and cookie notice if analytics, tracking pixels or marketing cookies are introduced.
- Avoid unsupported medical or treatment-result claims.
